# my-first-project
这是我的第一个 GitHub 项目，用来记录学习笔记和代码练习，同时包含我的个人介绍网页。

## 项目介绍
- 这是一个用 Trae 搭建的**个人介绍网页**，展示了我的基本信息、职业背景和联系方式。
- 技术栈：HTML + CSS，实现了响应式布局，适配手机和电脑端。
- 预览方式：直接下载 `index.html` 和 `style.css`，在浏览器中打开 `index.html` 即可查看效果。

## 学习计划
- 掌握 Git 基础操作
- 学习 GitHub 仓库管理
- 练习代码版本控制
- 学习用 AI 工具（Trae/Cursor）快速开发项目
